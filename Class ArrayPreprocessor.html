Class ArrayPreprocessor
Simple preprocessor that rewrites BOOSE array / poke / peek commands into equivalent scalar variable operations. This allows the supplied "unrestricted arrays" example program to run without relying on the restricted BOOSE.Array / Poke / Peek implementations.
Syntax
C#
    internal static class ArrayPreprocessor

Documentation Preview
Method RewriteArrays
Rewrites any occurrences of: array <type> <name> <size> poke <name> <index> = <expr> peek <var> = <name> <index> into combinations of normal int/real declarations and assignments.
Syntax
C#
    public static string RewriteArrays(string source)
Parameters
