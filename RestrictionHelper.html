Class RestrictionHelper
Provides helper methods to reset internal restriction counters and static limits in the BOOSE library, preventing "restriction limit reached" errors during long or nested program execution.
Syntax
C#
    public static class RestrictionHelper

Documentation Preview
Method ResetRestrictions
Resets counters and static limit fields across BOOSE commands and related types. Iterates through the BOOSE assembly and attempts to reset all static int fields. Specific obfuscated fields are also reset individually.
Syntax
C#
    public static void ResetRestrictions()
