Class AppWrite
Represents a command to write text on the canvas. Inherits from CommandOneParameter and provides functionality to display text messages with length restrictions.
Syntax
C#
    internal class AppWrite : CommandOneParameter
Remarks
This command requires exactly one parameter: the text message to be displayed. The text length is restricted to a maximum of 1000 characters.
